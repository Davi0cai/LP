# 🍔 Delivery System

## 📝 Sobre o Projeto
O **Delivery System** é um sistema completo de gerenciamento de restaurante e delivery desenvolvido em C++. Ele foi projetado com foco em Orientação a Objetos (POO), permitindo o controle de pedidos locais (mesas), gestão de entregas, cardápio (produtos e combos), controle de entregadores e processamento de pagamentos.

## ✨ Funcionalidades
- **Gestão de Cardápio:** Controle de produtos individuais e criação de "Combos" (agrupamento de produtos com quantidades específicas).
- **Atendimento de Mesas:** Controle de pedidos feitos por clientes que estão consumindo no estabelecimento.
- **Sistema de Entregas (Delivery):** Registro de pedidos para entrega domiciliar e associação a entregadores disponíveis.
- **Controle de Pagamentos:** Suporte a diferentes métodos de pagamento (Dinheiro, Pix, Cartão de Crédito).
- **Persistência de Dados:** Salvamento e carregamento automático de informações essenciais (cardápio, entregadores e pedidos) utilizando a classe de arquivos "Loader".

## Para executar corretamente:
- Código feito para ser executado no VS Code;
- Dados da pasta vscode a colocar em .vscode no projeto dentro do VS Code;
- Makefile deve estar fora das outras pastas;
- Siga as instruções nesse tutorial (https://code.visualstudio.com/docs/cpp/config-mingw) para preparar a máquina para execução;
- Compilação ideal usando ctrl + shift + b, e execução com f5;
- Se compilação der errado, tentar novamente ou usar o f5 parece resolver problemas (in)existentes; Usar make clean e make podem resolver possíveis erros também.
